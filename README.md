# 🎤 Голосовой помощник

Приложение для голосового общения с AI (Claude) — компаньон для пожилых людей с ограниченными возможностями зрения.

## 🚀 Возможности

- 🎙️ Непрерывное распознавание речи (Web Speech API)
- 🔊 Синтез речи (Google TTS в браузере)
- 💬 История диалога
- 🛡️ Защита от опасного контента (суицидальные идеи, наркотики)
- 🌍 Поддержка русского языка
- 📱 Работает на телефонах (PWA)
- 🔐 Защита паролем

## 📋 Требования

- Node.js 20+
- Claude API ключ (от Anthropic)

## 🛠️ Стек технологий

- **Frontend:** Vanilla JavaScript, HTML5, CSS3
- **Backend:** Vercel Serverless Functions (Node.js)
- **AI:** Claude API (Haiku 3.5)
- **Хостинг:** Vercel
