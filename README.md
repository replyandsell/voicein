# Voice Chat Companion

Голосовой помощник на базе Claude AI. Человек разговаривает голосом, AI отвечает голосом.

## Возможности

- 🎤 Непрерывное распознавание речи (Web Speech API)
- 🗣️ Синтез речи (Google TTS в браузере)
- 🇷🇺 Поддержка русского языка
- 💬 История диалога
- ⚠️ Фильтр на опасный контент (суицидальные идеи)
- 🔊 Регулировка скорости речи
- 📱 Работает на Android, iOS, ПК

## Технология

- **Frontend**: HTML5, JavaScript (Web Speech API)
- **Backend**: Vercel Functions (Node.js)
- **AI**: Claude API (Anthropic)

## Установка и развёртывание

### 1. Подготовка GitHub репозитория

```bash
# Клонируешь репо
git clone https://github.com/replyandsell/voice-companion.git
cd voice-companion

# Создаёшь ветку (опционально)
git checkout -b main
```

### 2. Добавление файлов

Скопируй в проект:
- `index.html` → в корень
- `api/chat.js` → создай папку `api/` и положи файл туда
- `package.json` → в корень
- `.gitignore` → в корень

Структура должна быть:
```
voice-companion/
├── api/
│   └── chat.js
├── index.html
├── package.json
├── .gitignore
└── README.md (этот файл)
```

### 3. Загрузка на GitHub

```bash
git add .
git commit -m "Initial commit: add voice chat companion"
git push origin main
```

### 4. Развёртывание на Vercel

1. Заходишь на vercel.com
2. Нажимаешь "New Project"
3. Выбираешь репо `voice-companion`
4. **ВАЖНО**: В Environment Variables добавляешь:
   - Имя: `CLAUDE_API_KEY`
   - Значение: твой API ключ Claude
5. Нажимаешь "Deploy"

Готово! Ссылка на приложение будет вроде: `https://voice-companion-xxx.vercel.app`

### 5. Использование

- Открываешь ссылку на Android в Chrome
- Нажимаешь большую кнопку "ГОВОРИ"
- Говоришь
- Слушаешь ответ
- Повторяешь

Для остановки разговора: говоришь "пока", "конец" или "спасибо".

## Для нескольких пользователей

Если хочешь чтобы приложение использовали разные люди (каждый платит свои токены):

1. Создай разные API ключи для каждого пользователя
2. Развёрни приложение несколько раз на Vercel (для каждого свой проект)
3. Каждому дай свою ссылку

Или используй одну ссылку, но разные ключи вводят через форму (сложнее).

## Устранение проблем

**Нет звука:**
- Проверь громкость телефона
- Разрешил ли браузер использовать микрофон

**Не распознаёт голос:**
- Говори чётче и громче
- Проверь интернет
- Попробуй русский язык (может быть выбран английский)

**API ошибка:**
- Проверь что API ключ добавлен в Vercel Environment Variables
- Проверь что у ключа есть кредиты на аккаунте Anthropic

## Безопасность

- API ключ хранится в Vercel, не видно в коде
- Браузер отправляет текст на `/api/chat`
- Backend запрашивает Claude API с ключом
- Никакие данные не логируются
